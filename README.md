# Trading GF

Trading GF is a React + TypeScript + Vite app for tracking open/closed trades, exits, performance, goals, and insights.

## Typography System

The app uses a three-level hierarchy:

### Primary (Hero)
- Size: 24-28px
- Weight: 700
- Color: `var(--text-strong)` (`#f5f8ff`)
- Use: Main metrics, P&L hero values, key headline numbers

### Secondary (Supporting)
- Size: 16-18px
- Weight: 600
- Color: `var(--text)` (`#e7eefb`)
- Use: Section headings, symbols, important supporting values

### Tertiary (Labels/Metadata)
- Size: 12-13px
- Weight: 500
- Color: `var(--muted)` (`#91a3c0`)
- Use: Labels, helper text, metadata, contextual descriptions

### Numeric Alignment
- Always apply `.text-numeric` to numbers to enable tabular numerals for cleaner vertical alignment.
